# Bike_Sales-Dashboard
The Bike Sales Dashboard is an interactive data visualization project designed to analyze customer demographics and purchasing behavior in a bike sales dataset. It provides clear insights into factors that influence bike purchases, helping businesses make data-driven decisions.
